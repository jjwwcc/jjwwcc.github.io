# 写一段markdown示范
Ask:
写一段markdown示范

Answer:
好的，下面是一段markdown示范：

```
# 这是标题
## 这是二级标题
### 这是三级标题

这里是正文内容

- 这是一条列表
- 这是另一条列表

这是一个链接：[百度一下](http://www.baidu.com)

这是引用：
&gt; 这是一段引用内容

```

 Ask:
来一段英文的

Answer:                            
Sure! Here is a markdown example in English:

```
# Title
## Subtitle
### Sub-subtitle

This is the main text.

- This is a list item
- This is another list item

This is a link: [Google it](http://www.google.com)

This is a quote:
> This is a quoted text

```
